How to Import from Ultimate Foreman (or any system)

  1. Export your data from your current system as CSV or Excel files
  2. Select the type of data you're importing (Jobs, Workers, Daily Records, or Concrete)
  3. Upload your file - the system will auto-detect column mappings
  4. Review and adjust the column mappings if needed
  5. Preview the data and click Import to bring it into the system
